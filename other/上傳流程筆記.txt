GitHub Pages 正常上傳流程
用git bash不要用cmd 還要載git

進入專案資料夾
cd "C:\Users\user\Desktop\YL-blog"

加入全部變動檔案
git add .

建立一個 commit 訊息
git commit -m "實作全域持續播放音樂系統"

推到 GitHub
git push origin main


遇到殘檔解決方式
# 1.刪除 Git 歷史（注意：會清空整個 Git 記錄）
rm -rf .git

# 2. 重新初始化 Git 專案
git init

# 3. 加入 GitHub 遠端倉庫
git remote add origin https://github.com/symboo/YL-blog.git

# 4. 加入你要的檔案（確認不要再放入太大的影片）
git add .

# 5. 提交訊息
git commit -m "重新上傳乾淨版網站"

# 6. 強制推送（因為 GitHub 上原本有歷史，要覆蓋掉）
git push -f origin main

git push -f origin master:main
